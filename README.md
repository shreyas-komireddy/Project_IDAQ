# SBC IDAQ Board - Integrated Data Acquisition System

## Overview
The **SBC IDAQ Board** is a fully integrated **Data acquisition system** designed for **high-speed analog and digital signal processing**.  
It features **16-bit and 14-bit ADCs** for precision data capture and also includes an on-board **DAC** for analog output.
The acquired data can be transmitted to the **application layer** through multiple high-speed peripherals — **UART**, **Ethernet**, and **USB** — allowing flexible downstream utilization for real-time monitoring, control systems, or data analytics.
This project embodies a full-stack embedded acquisition system — from signal sampling to high-speed data communication — designed and implemented at the hardware, firmware, and HDL levels.
